# TODO LIST 만들기

# 사용방법

1. Git Clone
   ```
   https://github.com/JongMany/KaKao-TodoList.git
   ```
2. 의존성 설치
   ```
   pnpm install
   ```
3. 실행
   ```
   pnpm run dev
   ```
